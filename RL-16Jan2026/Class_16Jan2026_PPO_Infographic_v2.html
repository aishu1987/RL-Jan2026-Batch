Scaling Reinforcement Learning with Neural Networks & PPO

Tabular RL → neural function approximation → stable policy optimization via PPO (clipping + value baseline + GAE).
Class Date: 16 Jan 2026
Theme: Stable learning at scale

Neural Networks (Function Approx.)

  • Generalize from continuous features
  • Replace Q-table with Qθ(s,a) or πθ(a|s)
Qθ(s,a) ≈ Q(s,a) • πθ(a|s)=softmax(fθ(s))

Actor–Critic

  • Actor: πθ(a|s)
  • Critic: Vϕ(s) baseline
  • Advantage drives updates
A(s,a)=Q(s,a)−V(s)

PPO (Clipping)

  • Use ratio r = πθ/πold
  • Clip to limit update size
  • Add entropy for exploration
Lclip=E[min(rA, clip(r,1−ε,1+ε)A)]
